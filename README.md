# Pick-e-flick 
A deep learning used Movie recommending system for cinephiles.

## features : 
- Content-based movie recommendations
- Movie embeddings using PyTorch
- FastAPI backend
- Interactive web interface
- Similar movie search

## Tech Stack

- Python
- PyTorch
- FastAPI
- Pandas
- NumPy
- HTML/CSS/JavaScript
